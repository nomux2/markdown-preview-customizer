# Changelog

All notable changes to the "Markdown Preview Customizer" extension will be documented in this file.

## [0.1.0] - 2025-12-26

### Added
- **Initial Beta Release (Renamed from Antigravity)**.
- **30+ Premium Themes**: Diverse styles across Business, Technical, Creative, and Design categories.
- **Slideshow Mode**: Interactive presentation mode with navigation and overview.
- **Advanced Exports**: Support for PDF, Word (.docx), and HTML (with Base64 resource embedding option).
- **Chart & Diagram Support**: Integrated Mermaid and Chart.js rendering.
- **Math Support**: High-performance LaTeX math rendering via KaTeX.
- **Code Block Features**: Improved syntax highlighting and "Copy to Clipboard" functionality.
- **Localization**: Full support for English and Japanese.
- **Sync Features**: Real-time scrolling and cursor synchronization between editor and preview/slideshow.
- **Live Update**: Optional toggle for live preview updates.
