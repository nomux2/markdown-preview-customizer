## üõ†Ô∏è How to Use

1. **Open Preview**: Click the preview icon in the editor title bar or press `Ctrl+Shift+V` (standard) or use the command `Markdown Preview: Open Preview to the Side`.
2. **Switch Themes**: Right-click in the preview window or use the Command Palette (`Ctrl+Shift+P`) and search for `Markdown Preview: Set Theme`.
3. **Start Slideshow**: Click the play icon in the editor title bar or use the context menu.
4. **Export**: Use the editor context menu (right-click in the Markdown file) to choose your export format.
