## 🇬🇧 English

**Markdown Preview Customizer** enhances your VS Code Markdown preview experience with professional themes, a powerful slideshow mode, and versatile export options.

### ✨ Key Features

*   **🎨 30+ Professional Themes**: Instantly apply beautiful styles for Design, Business, Technical, and Creative use cases.
*   **📽️ Interactive Slideshow Mode**: Turn your Markdown into a presentation with one click.
*   **📄 Pro Export Capabilities**: Export to PDF, Word (.docx), and HTML (Folder or Single File).
*   **📊 Advanced Components**: Native support for Mermaid diagrams, Chart.js, KaTeX, and Code Block copy button.

### 🎨 Themes
Choose from over 30 themes categorized for your needs:
*   **Design System**: Modern, Trust, Simple, Nordic, Elegant, Minimal.
*   **Business**: Corporate, Report, Minutes, Proposal, Contract.
*   **Creative & Technical**: Manuscript, Neon, Terminal.
*   **Presentation**: Keynote, Pitch, Whiteboard.

> **How to use**: Right-click in the preview or use command `Markdown Preview: Set Theme`.

### 📽️ Slideshow Mode
Separate slides with `---` (horizontal rule).
*   **Start**: Click the "Play" icon in the editor title bar.
*   **Navigate**: Use Arrow keys or Spacebar.
*   **Overview**: Press `Esc` to see all slides.
*   **Sync**: Cursor position in editor syncs with the current slide.

### 📄 Export
Right-click in the Markdown editor to access export commands:
*   **PDF**: Print-ready output.
*   **Word**: Editable .docx file.
*   **HTML**: Export as a standalone file (Base64) or folder structure.

## 🇯🇵 日本語 (Japanese)

**Markdown Preview Customizer** は、VS CodeのMarkdownプレビューを強力に拡張するツールです。美しいテーマ、スライドショー機能、そして高度なエクスポート機能を提供します。

### ✨ 主な機能

*   **🎨 30種類以上のプロフェッショナルテーマ**: デザイン、ビジネス、技術文書、クリエイティブなど、用途に合わせた美しいテーマを即座に適用できます。
*   **📽️ スライドショーモード**: Markdownファイルをワンクリックでインタラクティブなプレゼンテーションに変換します。
*   **📄 高度なエクスポート**: PDF、Word (.docx)、HTML（フォルダ形式または単一ファイル）への書き出しに対応しています。
*   **📊 拡張コンポーネント**: Mermaid記法、Chart.js、数式（KaTeX）、コードブロックのコピーボタンを標準サポート。

### 🎨 テーマの変更方法
プレビュー画面で**右クリック**するか、コマンドパレットから `Markdown Preview: Set Theme` を実行してください。以下のカテゴリーから選択できます。

*   **Design System (デザイン)**: Modern, Trust, Simple, Nordic, Elegant, Minimal など
*   **Business (ビジネス)**: Corporate, Report, Minutes (議事録), Proposal (提案書), Contract (契約書) など
*   **Technical (技術)**: Tech Manual, Terminal, Neon など
*   **Presentation (プレゼン)**: Keynote, Pitch, Whiteboard など

### 📽️ スライドショーの使い方
Markdown内で `---` (水平線) を記述すると、そこがスライドの区切りになります。

1.  **開始**: エディタ右上の「再生ボタン」アイコンをクリック、またはコンテキストメニューから「スライドショーを開始」を選択します。
2.  **操作**: 矢印キーまたはスペースキーでページ送りできます。
3.  **一覧表示**: `Esc` キーを押すと、全スライドを一覧表示（オーバービュー）できます。
4.  **同期**: エディタのカーソル位置とスライドの表示が同期します。

### 📄 エクスポート（PDF / Word / HTML）
Markdownエディタ上で**右クリック**し、メニューから出力形式を選択してください。

*   **PDF**: 印刷や共有に最適なPDF形式で出力します。
*   **Word**: 編集可能な Word (.docx) 形式で出力します。
*   **HTML**: 
    *   **Folder**: 画像やCSSを別フォルダに分けて出力します。
    *   **Base64**: すべてのリソースを1つのHTMLファイルに埋め込んで出力します（配布に便利です）。

## ⚙️ 設定 (Configuration)

VS Codeの設定画面 (`Ctrl+,`) から以下の項目をカスタマイズできます。

| Setting / 設定名 | Description / 説明 |
| :--- | :--- |
| `markdownPreviewCustomizer.theme` | 起動時のデフォルトテーマを設定します。<br>Set the default theme on startup. |
| `markdownPreviewCustomizer.showDebugLog` | プレビュー画面にデバッグログを表示します（開発者用）。<br>Show debug log overlay in preview. |
